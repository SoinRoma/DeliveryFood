<h1 align="center">Delivery Food</h1>

## Описание
Многостраничный сайт для заказа еды из разных кафе и ресторанов.
На сайте реализовано:
+ База данных ресторанов
+ Корзина для добавления
+ Адаптив
+ Анимация
